# OpenCV 切痕检测预处理方案说明

## 📋 概述

本模块提供了10种不同的OpenCV预处理方案，用于改进晶圆切痕检测的准确性和鲁棒性。每种方案针对不同的图像条件进行了优化。

---

## 🎯 预处理方案列表

### 方案1: 直方图均衡化 (Histogram Equalization)
**适用场景**: 图像整体对比度低，但切痕与背景有灰度差异

**优点**:
- ✅ 增强整体对比度
- ✅ 简单快速
- ✅ 适合光照不均的情况

**缺点**:
- ❌ 可能过度增强噪声
- ❌ 不适合对比度已经很好的图像

**使用方法**:
```python
from preprocessing_methods import preprocess_method_1_histogram_equalization
processed = preprocess_method_1_histogram_equalization(image)
```

---

### 方案2: CLAHE自适应直方图均衡化 ⭐推荐
**适用场景**: 局部光照不均，需要保持局部细节

**优点**:
- ✅ 限制局部对比度增强，避免过度增强
- ✅ 保持图像自然外观
- ✅ 适合有阴影或光照变化的图像

**参数**:
- `clip_limit`: 对比度限制阈值 (默认2.0, 范围1.0-4.0)
- `tile_size`: 局部区域大小 (默认(8,8), 可选(4,4), (8,8), (16,16))

**使用方法**:
```python
from preprocessing_methods import preprocess_method_2_clahe
processed = preprocess_method_2_clahe(image, clip_limit=2.0, tile_size=(8, 8))
```

---

### 方案3: 自适应阈值 (Adaptive Threshold)
**适用场景**: 光照不均，全局阈值效果差

**优点**:
- ✅ 自动适应局部光照变化
- ✅ 对光照不均的图像效果好
- ✅ 直接输出二值图像

**参数**:
- `block_size`: 局部区域大小 (必须为奇数，如11, 15, 21)
- `C`: 从均值中减去的常数 (通常2-5)
- `method`: `cv.ADAPTIVE_THRESH_MEAN_C` 或 `cv.ADAPTIVE_THRESH_GAUSSIAN_C`

**使用方法**:
```python
from preprocessing_methods import preprocess_method_3_adaptive_threshold
processed = preprocess_method_3_adaptive_threshold(image, block_size=11, C=2)
```

---

### 方案4: 形态学边缘检测 (Morphology Edge Detection)
**适用场景**: 需要突出切痕边缘，去除噪声

**优点**:
- ✅ 突出边缘特征
- ✅ 对噪声有抑制作用
- ✅ 适合边缘清晰的切痕

**参数**:
- `kernel_size`: 形态学核大小 (3, 5, 7)
- `morph_type`: 
  - `"gradient"`: 形态学梯度（突出边缘）
  - `"tophat"`: 顶帽（突出比周围亮的区域）
  - `"blackhat"`: 黑帽（突出比周围暗的区域，**适合暗色切痕**）

**使用方法**:
```python
from preprocessing_methods import preprocess_method_4_morphology_edge
# 形态学梯度
processed = preprocess_method_4_morphology_edge(image, morph_type="gradient")
# 黑帽（推荐用于暗色切痕）
processed = preprocess_method_4_morphology_edge(image, morph_type="blackhat")
```

---

### 方案5: 高斯滤波 + Canny边缘检测
**适用场景**: 需要精确的边缘定位，噪声较多

**优点**:
- ✅ 边缘定位精确
- ✅ 噪声抑制效果好
- ✅ 适合需要精确边缘的场景

**参数**:
- `gaussian_kernel`: 高斯核大小 (必须为奇数，如5, 7, 9)
- `canny_low`: Canny低阈值 (默认50)
- `canny_high`: Canny高阈值 (默认150)

**使用方法**:
```python
from preprocessing_methods import preprocess_method_5_gaussian_canny
processed = preprocess_method_5_gaussian_canny(image, gaussian_kernel=5, 
                                               canny_low=50, canny_high=150)
```

---

### 方案6: 对比度和亮度调整
**适用场景**: 图像整体偏暗或对比度不足

**优点**:
- ✅ 可控的对比度和亮度调整
- ✅ 简单直接
- ✅ 适合微调图像质量

**参数**:
- `alpha`: 对比度控制 (1.0-3.0, 1.0为原始)
- `beta`: 亮度控制 (-100到100, 0为原始)

**使用方法**:
```python
from preprocessing_methods import preprocess_method_6_contrast_enhancement
processed = preprocess_method_6_contrast_enhancement(image, alpha=1.5, beta=30)
```

---

### 方案7: 双边滤波 (Bilateral Filter) ⭐推荐
**适用场景**: 需要去噪但保持边缘清晰

**优点**:
- ✅ 去噪同时保持边缘
- ✅ 适合有噪声但边缘重要的图像

**参数**:
- `d`: 邻域直径 (通常5-15)
- `sigma_color`: 颜色空间标准差 (通常50-100)
- `sigma_space`: 坐标空间标准差 (通常50-100)

**使用方法**:
```python
from preprocessing_methods import preprocess_method_7_bilateral_filter
processed = preprocess_method_7_bilateral_filter(image, d=9, 
                                                  sigma_color=75, sigma_space=75)
```

---

### 方案8: Otsu自动阈值
**适用场景**: 双峰直方图，需要自动确定最佳阈值

**优点**:
- ✅ 自动确定最佳阈值
- ✅ 适合双峰分布的图像
- ✅ 无需手动调整阈值

**使用方法**:
```python
from preprocessing_methods import preprocess_method_8_otsu_threshold
binary, threshold_value = preprocess_method_8_otsu_threshold(image)
print(f"自动阈值: {threshold_value}")
```

---

### 方案9: 多尺度处理
**适用场景**: 切痕宽度变化大，需要多尺度检测

**优点**:
- ✅ 适应不同宽度的切痕
- ✅ 提高检测鲁棒性

**使用方法**:
```python
from preprocessing_methods import preprocess_method_9_multi_scale
scaled_images = preprocess_method_9_multi_scale(image, scales=[1.0, 0.5, 2.0])
```

---

### 方案10: 组合预处理 ⭐推荐
**适用场景**: 复杂图像条件，单一方法效果不佳

**优点**:
- ✅ 结合多种方法优势
- ✅ 适应复杂场景

**使用方法**:
```python
from preprocessing_methods import preprocess_method_10_combined
processed = preprocess_method_10_combined(image, 
    method_sequence=["clahe", "bilateral", "morphology"])
```

## 📊 方案选择指南

| 图像条件 | 推荐方案 | 备选方案 |
|---------|---------|---------|
| **正常图像** | CLAHE | 直方图均衡化 |
| **低对比度** | CLAHE + 对比度增强 | 直方图均衡化 |
| **噪声较多** | 双边滤波 + 形态学 | 高斯滤波 + Canny |
| **光照不均** | 自适应阈值 | CLAHE |
| **整体偏暗** | 对比度增强 + CLAHE | 直方图均衡化 |
| **有反射干扰** | 黑帽形态学 + 双边滤波 | 自适应阈值 |
| **边缘不清晰** | 形态学梯度 | Canny边缘检测 |
| **复杂场景** | 组合方法 | 多尺度处理 |

## 💡 使用建议

### 1. 先分析图像
使用 `analyze_image_statistics()` 函数分析图像统计信息，了解图像特点。

### 2. 尝试推荐方案
使用 `get_recommended_preprocessing()` 让系统自动推荐预处理方法。

### 3. 对比效果
使用 `preprocessing_comparison.py` 对比不同方法的效果，选择最佳方案。

### 4. 参数调优
根据实际效果调整各方法的参数，特别是：
- CLAHE的 `clip_limit` (1.5-3.0)
- 自适应阈值的 `block_size` (11-21) 和 `C` (2-5)
- 双边滤波的 `sigma_color` 和 `sigma_space` (50-100)

### 5. 组合使用
对于复杂场景，可以组合多个方法：
```python
# 先CLAHE增强对比度
result = preprocess_method_2_clahe(image)
# 再双边滤波去噪
result = preprocess_method_7_bilateral_filter(result)
# 最后形态学突出边缘
result = preprocess_method_4_morphology_edge(result, morph_type="gradient")
```

---

## 📝 注意事项

1. **预处理顺序很重要**: 通常先增强对比度，再去噪，最后提取边缘
2. **参数需要调优**: 不同图像可能需要不同的参数，建议先使用默认值，再根据效果调整
3. **计算开销**: 组合方法会增加计算时间，需要平衡效果和性能
4. **ROI区域**: 预处理应该应用到ROI区域，而不是整张图像，以提高效率

## 🔧 常见问题

**Q: 哪种方法最适合我的图像？**  
A: 使用 `analyze_image_statistics()` 分析图像，然后根据推荐选择。通常CLAHE和双边滤波的组合效果较好。

**Q: 预处理后如何调整检测阈值？**  
A: 预处理改变了图像特性，可能需要重新计算 `image_mean` 或调整阈值策略。

**Q: 预处理会影响检测速度吗？**  
A: 是的，特别是组合方法。建议只在ROI区域应用预处理，而不是整张图像。

**Q: 如何保存预处理结果？**  
A: 使用 `cv.imwrite()` 保存处理后的图像，或使用matplotlib保存对比图。
